// 01. THE MISSION

**PostQuantu CLI** is your dedicated interface for migrating today's data and communications to a quantum-resistant state. Developed using standard **NIST algorithms**, it ensures your security remains unbreakable when large-scale quantum computers become a reality.

PQC Algorithms

Implements Kyber (Key Exchange) and Dilithium (Digital Signatures), the leading post-quantum cryptography standards selected by NIST.

Digital Signatures

Create unforgeable digital signatures for files and documents, ensuring integrity and authenticity in a quantum era.

Key Management

Tools for easy generation, storage, and exchange of quantum-resistant keys without complex configurations.

// 02. HOW TO USE

Once installed via the native package, interact with the CLI directly from your terminal. Here are common commands:

postquantu postquantu keygen --alg dilithium --output my_dilithium_key Keys successfully generated: my_dilithium_key.pub, my_dilithium_key.priv postquantu sign --file document.txt --key my_dilithium_key.priv Signature file created: document.txt.sig
